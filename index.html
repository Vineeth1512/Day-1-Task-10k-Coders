Html:

HTML stands for Hypertext markup language

It is used to create webpages and it provides the structure of a webpage using elements represented by tags.

CSS:

CSS Means Cascading Style Sheets

CSS defines how HTML elements are to be displayed. It helps to format and add styles to HTML elements.

JavaScript:

JavaScript is a lightweight, cross-platform, object-oriented programming language.

HTML is used to show content on a webpage, and CSS is used to add styles to that content and handle the layout. JavaScript is used to add interactivity and dynamic effects to that content.

React.js:

React is a JavaScript library for building Single-Page Applications with interactive user interfaces where the UI dynamically updates without requiring a full page reload.

MongoDb:

MongoDB is a NoSQL, document-oriented database that is designed to store and retrieve large volumes of data efficiently.

It is highly scalable, flexible, and widely used for building modern applications. MongoDB stores data in JSON-like documents with dynamic schemas, making it a popular choice for applications requiring a flexible data structure.

Node Js:

Node.js is an open-source, cross-platform, JavaScript runtime environment built on Google Chrome's V8 JavaScript engine. It enables developers to execute JavaScript code outside the browser, making it ideal for server-side and network applications.

Git:

git is a distributed version control system that helps track changes to our code collaborate with others and manage project versions.
